## 1. Overview

The **App Management Centre (AMC)** is the central governance and operational oversight platform for the Maturion system. It provides CS2 with real-time visibility and control over all governance-sensitive operations.

**Previously known as**: Foreman App (`maturion-foreman-app`)  
**New repository target**: `maturion-amc`

---

## 2. Core Responsibilities

1. **ARC Trigger Governance** — Human-in-the-loop review for all ARC triggers before GitHub issue creation
2. **Dynamic Upload Quota Management** — Real-time quota monitoring and adjustment
3. **Alert Dashboard** — Centralised system alerts with threshold notifications
4. **Audit Trail** — Tamper-evident, exportable log of all governance decisions
5. **Progressive Automation Control** — CS2-controlled automation level settings
