## Definition of Ready checklist (mandatory)
- [ ] RED tests allocated and reconciled.
- [ ] Required routes/screens/states listed.
- [ ] Required data/API/RLS changes listed.
- [ ] Required audit events listed.
- [ ] Required notification events listed.
- [ ] Required denied-path tests listed.
- [ ] Required LFV/deployed evidence listed.
- [ ] Dependencies and sequencing confirmed.
- [ ] Rollback plan recorded.
- [ ] No unresolved appointment/build blocker open.
- [ ] Build Authorization explicitly cleared before execution.
