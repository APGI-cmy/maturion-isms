# MAT MODULE --- APP DESCRIPTION v2.0

**Status:** Draft\
**Owner:** Johan Ras\
**Date:** 2026-02-20\
**Module Role:** Manual Audit Execution Engine\
**Interoperability:** Roadmap, RADAM, PIT

------------------------------------------------------------------------

## 1. Purpose

MAT enables structured, defensible manual audits aligned with the
Roadmap architecture.

------------------------------------------------------------------------

## 2. Structural Alignment (Mandatory)

Hierarchy identical to Roadmap:

Domain → MPS → Criteria

Numbering immutable after approval.

------------------------------------------------------------------------

## 3. Criteria Creation Modes

A)  Import from Roadmap\
B)  Document import + AI restructuring

AI must not invent criteria.

------------------------------------------------------------------------

## 4. Evidence Modal

Each criterion supports:

-   Findings
-   Voice recording
-   Photo capture
-   Video capture
-   File upload
-   Interview transcript

Offline mode supported.

------------------------------------------------------------------------

## 5. Maturity Model

Identical 5-level model as Roadmap.

Scoring includes:

-   Proposed level
-   Confidence score
-   Rationale
-   Gaps (immediate/medium/long)

Human confirmation mandatory.

------------------------------------------------------------------------

## 6. Outputs

1)  PDF/DOCX report\
2)  Structured JSON export to Roadmap\
3)  PIT Draft Plan JSON

------------------------------------------------------------------------

## 7. PIT Integration Hooks

From findings and gaps, AI generates structured implementation plan:

Project → Phase → Task → Subtask

Each task must reference criterion number.

Exportable even before PIT exists.

------------------------------------------------------------------------

## 8. AI Governance

-   Confidence scoring
-   Human override logging
-   Model version tracking
-   Override analytics

------------------------------------------------------------------------

## 9. Strategic Position

MAT = Field audit layer for lower maturity stages. As maturity
increases, RADAM replaces manual auditing.

------------------------------------------------------------------------

End of Document
