## Usage

Copy this template to:

```
.agent-workspace/<agent>/memory/liaison-ceremony-<wave>-YYYYMMDD.md
```

Fill every section. Do NOT leave placeholder values. This document is append-only after commitment — do not edit a committed copy; create a new copy for subsequent rounds.

This pack is required for any governance-liaison or non-ECAP flow that produces final-state artifacts (session memories, inventory updates, wave records, or PREHANDOVER proofs) where the ECAP reconciliation stack is not active.
