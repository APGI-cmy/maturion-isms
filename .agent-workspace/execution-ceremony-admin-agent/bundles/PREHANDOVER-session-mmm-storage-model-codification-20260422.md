- Reads use `authenticated` role. `mmm_evidence_org_read_v2` enforces org-level SELECT isolation.

### FK / Dependency Chain

- `mmm-evidence` bucket already created by `20260420000004_mmm_storage_buckets.sql`. Audio MIME fix uses `UPDATE ... WHERE id = 'mmm-evidence'` — no FK dependency.
