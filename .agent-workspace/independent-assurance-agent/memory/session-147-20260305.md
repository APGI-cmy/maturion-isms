1. **CodexAdvisor PREHANDOVER template gap — Pre-IAA Commit Gate**: CodexAdvisor's PREHANDOVER template does not include an explicit Pre-IAA Commit Gate step requiring `git status` and `git log --oneline -3` evidence. Four consecutive sessions (143, 144, 145, now 147) have seen A-021 commit failures from different agents. The PREHANDOVER template for CodexAdvisor-agent should include: `## Pre-IAA Commit Gate` with explicit `git status` output showing no untracked/unstaged deliverables, and `git log --oneline -3` showing the commit containing all declared artifacts. This would structurally prevent A-021 violations at source.

Recommend CodexAdvisor-agent's template include this section with a default boilerplate: `NO DOWNSTREAM RIPPLE REQUIRED / [list affected files]`. This is the first AGENT_CONTRACT PR from CodexAdvisor to omit this section — codifying it in the template prevents recurrence.

This matches the session-142 suggestion and is now confirmed as a recurring need.
