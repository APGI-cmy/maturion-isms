## Suggestions for Improvement (MANDATORY)

**Suggestion 1** (highest priority — prevents OVL-AC-011 regression): Add explicit dual-column requirement to CodexAdvisor's PREHANDOVER template: `| File | Chars Before | Chars After | Delta | Limit | Status |`. This forces "before" population at template time. CodexAdvisor can get "before" from `git show HEAD~1:<path> | wc -m` before making changes. A pre-populated template prevents this class of failure entirely.
