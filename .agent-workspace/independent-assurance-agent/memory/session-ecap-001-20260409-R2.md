## Learning Notes

**L-001 — Administrator-class contract pattern established:**
The `execution-ceremony-admin-agent` establishes the first administrator-class agent contract
in the system. Key characteristics: PHASE 4 entirely delegated to Foreman (ceremony-admin has
no Phase 4 authority), two CONSTITUTIONAL prohibitions (NO-IAA-INVOCATION-001 and SELF-MOD-ECA-001),
and explicit HALT-001 for cases where Foreman has not accepted substantive readiness. This
pattern should be reused for any future administrator-class agents.

**L-002 — Pre-existing character count breach not introduced by PR:**
The `independent-assurance-agent.md` was already over the 30,000-char limit on main (30,580 chars)
before this PR. The PR's stated intent was compression — and achieved it for the prohibitions
and HALT sections — but new ECAP-001 constitutional content offset the reduction.
Future IAA reviewers: this is NOT a new finding introduced by ECAP-001. Check the main branch
char count before treating this as a new PR failure.

**L-003 — S-033 exception pattern works correctly for additive-only CI changes:**
The S-033 Inherent Limitation Exception with 3 substitutes (risk assessment + structural
correctness + CI run evidence) was correctly applied for the agent-contract-audit.yml change.
This is the right pattern for purely additive CI array changes where logic is unchanged.

**L-004 — HFMC-02 parity with meta-files:**
SCOPE_DECLARATION.md correctly handles the circular reference issue by not listing itself in
its own scope. The A-031 carve-out section explicitly documents the exclusions. IAA should
accept the SCOPE_DECLARATION.md self-update as an implicit inclusion, not a parity failure.

## Suggestions for Improvement

**S-001 — CS2 governance review for constitutional contract character limits:**
The 30,000-char limit was designed for operational contracts (builders, specialists). Constitutional
contracts (IAA, Foreman) legitimately require more text to define their governance obligations.
CS2 should consider whether a separate character limit (e.g., 35,000) should apply to
constitutional-class contracts, or whether a formal compression wave targeting IAA Phase 0-4
sections (e.g., consolidating repetitive output format declarations) would achieve the goal.
